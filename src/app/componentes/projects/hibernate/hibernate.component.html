¿En qué consiste este CRUD?

CRUD Música Hibernate

Para la realización de este CRUD se utilizaron los siguientes lenguajes: Java, Hibernate y JavaServer Faces asi como el gestor de bases de datos MySQL.

Este CRUD está realizado en base a un repertorio de Música, en donde el usuario podrá agregar a los artistas asi como los grupos de Música, los Albumes y las Canciones del mismo, asi como modificarlos y finalmente eliminarlos.

Compilado

Para poder compilar este proyecto, primero deberás contar algun editor de código (de preferencia NetBeans), asi como un gestor de bases de Datos.

Primero se deberá modificar el Archivo de hibernate.cfg.xml que se encuentra en la carpeta de config.


Ahora para poder compilar el proyecto, se dará clic en el botón de la flecha verde o en su defecto con la tecla F6.

Manejo del Programa

Para poder insertar registros, se podrá dirigir al apartado de acciones y posteriormente seleccionar la opción que quiera insertar.

Lista

Para poder visualizar los registros insertados se podrá dirigir al apartado de Lista y posteriormente seleccionar la lista que desea visualizar.
