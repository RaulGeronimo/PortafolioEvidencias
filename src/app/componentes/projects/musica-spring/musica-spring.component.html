¿En qué consiste este CRUD?

CRUD Música Spring

Para la realización de este CRUD se utilizaron los siguientes lenguajes: Java, JSP y Spring MVC asi como el gestor de bases de datos MySQL.

Este CRUD está realizado en base a un repertorio de Música, en donde el usuario podrá agregar a los artistas asi como los grupos de Música, los Albumes y las Canciones del mismo, asi como modificarlos y finalmente eliminarlos.

Compilado

Para poder compilar este proyecto, primero deberás contar algun editor de código (de preferencia NetBeans), asi como un gestor de bases de Datos.

Primero se deberá modificar el Archivo de ConexionDB que se encuentra en la carpeta de config.


Ahora para poder compilar el proyecto, se dará clic en el botón de la flecha verde o en su defecto con la tecla F6.

Manejo del Programa

Para poder insertar registros, se podrá dirigir a cualquiera de los apartados que aparecen en el menú principal.

Lista

Para poder visualizar los registros insertados se podrá dirigir al apartado de Lista que se encuentra en cada una de las secciones del menú principal. Asimismo, en cada una de las tablas se encontrará en la parte superior una serie de botones que servirán para exportar los registros.
