¿En qué consiste este CRUD?

CRUD Restaurante PHP

Para la realización de este CRUD se utilizaron los siguientes lenguajes: HTML, JavaScript, CSS y PHP asi como el gestor de bases de datos MySQL.

Este CRUD está realizado en base a un restaurante de comida francesa, en donde el usuario podrá agregar una reservación asi como los pedidos de Entradas, Bebidas y Postres del mismo, asi como modificarlos y finalmente eliminarlos.

Compilado

Para poder compilar este proyecto, primero deberás contar algun editor de texto, asi como un gestor de bases de Datos.

Primero se deberá modificar el Archivo de Conexion.php que se encuentra en la carpeta de PHP.


Ahora para poder compilar el proyecto, se deberán activar los servicios de apache y MySQL en Xammp.


Finalmente para ejecutar el proyecto, se debera abrir el navegador y colocar la direccion localhost/Restaurante/. Antes de ejecutarlo se debera asegurar que el proyecto se encuentre en la carpeta C:\xampp\htdocs

Manejo del Programa

Para poder insertar registros, se podrá dirigir al menú y seleccionar Reservar o en la página principal desplazarse al apartado de Reservar.

Una vez insertado el Cliente, se da clic en el botón de Ordenar y posteriormente se elige el tipo de platillo asi como el Código del mismo (Para esto se deberá visitar la lista de los platillos que están separados por categorías en el apartado de carta).

Factura

Una vez terminado de ordenar, se dará clic en el botón de realizar factura y se selecciona el cliente, después de esto se generara un archivo PDF.
