¿En qué consiste este CRUD?

CRUD Repertorio de Videojuegos

Para la realización de este CRUD se utilizaron los siguientes lenguajes: HTML, PHP, JS, para el diseño se utilizo el framework de bootstrap asi como el gestor de bases de datos MySQL.

Este CRUD está realizado en base a un repertorio de Videojuegos, en donde el usuario podrá registrar a los Videojuegos, asi como las empresas, y finalmente el desarrollador.

Diagrama de Base de Datos

Este diagrama consiste en un total de 5 tablas, las cuales son: Desarrollador, Director, Distribuidor, Juegos y País. Este proyecto también cuenta con un total de 4 vistas asi como unos procedimientos almacenados.

Compilado

Para poder compilar este proyecto, primero deberás contar algun editor de texto, asi como un gestor de bases de Datos.

Primero se deberá modificar el Archivo de Conexion.php que se encuentra en la carpeta de PHP.


Ahora para poder compilar el proyecto, se deberán activar los servicios de apache y MySQL en Xammp.


Finalmente para ejecutar el proyecto, se deberá abrir el navegador y colocar la dirección http://localhost/Videojuegos/. Antes de ejecutarlo se debera asegurar que el proyecto se encuentre en la carpeta C:\xampp\htdocs

Manejo del Programa

Antes de querer insertar datos, el usuario deberá seleccionar la sección que va a querer insertar.

Una vez seleccionado el apartado, aparecerá un botón de Agregar.

Actualizar o eliminar

Una vez estando los registros, cada registro contara con un botón de Actualizar o Eliminar.
