¿En qué consiste este CRUD?

CRUD Aplicación NodeJS con SQL

Para la realización de este CRUD se utilizaron los siguientes lenguajes: Nodejs, MySQL, Express asi como Handlebars, para el diseño se utilizaron los framework de CSS y Bootstrap asi como el gestor de bases de datos MySQL.

Este proyecto esta en base a una aplicación de contactos y enlaces preferidos en donde el usuario podrá guardar su lista de contactos asi como los enlaces que visite habitualmente, siendo que estos registros no podrán ser visualizadas por otros usuarios.

Compilado

Para poder compilar este proyecto, primero deberás contar con Visual Studio Code, asi como un gestor de bases de Datos.

Primero se deberá modificar el Archivo de keys que se encuentra en la carpeta de src.


En una terminal se colocará el comando npm run dev. Esto también hara que se genere la tabla sessions. Para que funcione ya debe estar en el equipo la Base de Datos

Manejo del Programa

Para poder insertar registros, primero se debera registrar en la seccion de Registrar o dando clic desde el inicio en Empezar

Lista

Una vez que el usuario se encuentre registrado podra acceder, en la pantalla de perfil aparecera una bienvenida asi como un pequeño menú.

Contactos

Cuando el usuario ya haya creado algún contacto, en cada una de los contactos se encontrara un boton para actualizar o en su defecto para eliminar, estos contactos no las podran visualizar otros usuarios.

Enlaces

Cuando el usuario ya haya creado algún enlace, en cada una de los enlaces se encontrara un boton para actualizar o en su defecto para eliminar, estas notas no las podran visualizar los demas usuarios.
